# presidium-contrib

Adapters and reference implementations for [Presidium](https://github.com/civitas-io/presidium) governance.

```bash
pip install presidium-contrib[opa]            # OPA policy engine adapter
pip install presidium-contrib[openbao]         # OpenBao/Vault credentials (MPL 2.0)
pip install presidium-contrib[agentgateway]    # AgentGateway LLM + MCP routing
pip install presidium-contrib[slack]           # Slack approval service
pip install presidium-contrib[webhook]         # Webhook approval provider
pip install presidium-contrib[postgres]        # PostgreSQL agent registry
```

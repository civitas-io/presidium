# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Presidium, please report it responsibly.

**Email:** jerynmathew@gmail.com

**Please include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** Within 2 business days
- **Assessment:** Within 5 business days
- **Fix for CRITICAL/HIGH:** Within 14 days
- **Coordinated disclosure:** 90-day window

## Scope

- Presidium core packages (presidium-registry, presidium-policy, presidium-llm-gateway, presidium-mcp-gateway, presidium-eval, presidium-sdk)
- CI/CD configurations
- Documentation that could lead to insecure configurations

## Out of Scope

- Civitas runtime vulnerabilities (report to Civitas project)
- Third-party dependencies (report to the respective project)
- Social engineering attacks
